# Day 31 – 2026 Interactive Vision Portfolio ✨🎯

Part of my **100 Days Frontend Development Challenge**.

## 📌 Project Overview
A multi-section interactive personal portfolio representing my vision for 2026.

This project combines:
- 3D flip animations
- Image hover overlays
- Perspective effects
- Section-based storytelling
- Food flip cards
- Interactive split panels
- Smooth transitions

It reflects growth, reflection, celebration, and intention.
