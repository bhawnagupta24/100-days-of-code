# Day 63 – JavaScript Amazon Shipping Calculator 📦

Part of my **100 Days Frontend Development Challenge**.

## 📌 Project Overview

A simple JavaScript shipping calculator inspired by Amazon's checkout process.

Users can enter the cost of their order, and the application automatically calculates the final amount. If the order value is below **$40**, an additional **$10 shipping charge** is applied. Users can also press the **Enter** key to calculate the total instantly.
