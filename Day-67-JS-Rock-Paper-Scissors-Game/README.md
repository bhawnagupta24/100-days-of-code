# Day 67 – JavaScript Rock Paper Scissors Game 🎮✂️📄🪨

Part of my **100 Days Frontend Development Challenge**.

## 📌 Project Overview

Built a fully interactive Rock Paper Scissors game using JavaScript.

The project randomly generates the computer's move, compares it with the player's selection, displays the result instantly, and keeps track of wins, losses, and ties using Local Storage so the score remains available after refreshing the page.

## 🛠 Tech Stack

- HTML5
- JavaScript (ES6)
- DOM Manipulation
- Local Storage

## ✨ Features

- Rock, Paper, and Scissors buttons
- Random computer move generation
- Instant game result
- Live score updates
- Score persistence using Local Storage
- Reset score functionality
- Dynamic DOM updates

## 📚 Concepts Practiced

- Functions
- Conditional Statements
- Math.random()
- Objects
- Local Storage
- JSON.parse()
- JSON.stringify()
- querySelector()
- innerHTML
- Event Handling

## 🚀 Learning Outcome

Learned how to build an interactive JavaScript game by combining random logic, DOM manipulation, reusable functions, and Local Storage for persistent game data.
