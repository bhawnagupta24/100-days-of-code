# Day 68 – JavaScript Subscribe Button & Shipping Calculator 🔔🛒

Part of my **100 Days Frontend Development Challenge**.

## 📌 Project Overview

Built two mini JavaScript projects to practice DOM manipulation and event handling.

The first project simulates a YouTube Subscribe button that toggles between **Subscribe** and **Subscribed** states.

The second project is an Amazon Shipping Calculator that calculates the final order cost by adding shipping charges when the order value is below the free shipping limit.

## 🛠 Tech Stack

- HTML5
- JavaScript (ES6)
- DOM Manipulation

## ✨ Features

### YouTube Subscribe Button
- Toggle Subscribe/Subscribed
- Dynamic button text update
- Click event handling

### Amazon Shipping Calculator
- User order cost input
- Automatic shipping charge calculation
- Free shipping condition
- Enter key support
- Dynamic total cost display

## 📚 Concepts Practiced

- Functions
- DOM Manipulation
- querySelector()
- innerHTML
- Event Handling
- onClick Events
- Keyboard Events
- Conditional Statements
- Number Conversion

## 🚀 Learning Outcome

Learned how to create interactive JavaScript mini projects using DOM manipulation, conditional logic, user input handling, and keyboard events to improve user experience.
