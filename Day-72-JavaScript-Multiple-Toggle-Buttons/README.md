# Day 72 – JavaScript Multiple Toggle Buttons 🎮🎵💻

Part of my **100 Days Frontend Development Challenge**.

## 📌 Project Overview

Built a multiple toggle button interface using JavaScript and CSS.

This project allows users to independently toggle different category buttons such as **Gaming**, **Music**, and **Tech**. Each button changes its appearance when clicked by dynamically adding or removing a CSS class using JavaScript.

The project strengthens understanding of DOM manipulation, CSS class management, and interactive UI behavior.

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)

## ✨ Features

- Gaming toggle button
- Music toggle button
- Tech toggle button
- Independent toggle functionality
- Active/Inactive button states
- Dynamic CSS class switching
- Interactive UI
- Beginner-friendly implementation

## 📚 Concepts Practiced

- DOM Manipulation
- querySelector()
- classList
- classList.add()
- classList.remove()
- classList.contains()
- Event Handling
- Conditional Statements
- CSS Class Management

## 🚀 Learning Outcome

Learned how to build interactive category buttons by dynamically adding and removing CSS classes. This project improved my understanding of DOM manipulation and reusable UI interaction patterns in JavaScript.
