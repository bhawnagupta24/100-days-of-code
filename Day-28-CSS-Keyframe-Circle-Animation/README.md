# Day 28 – CSS Keyframe Circle Animation ⚫✨

Part of my **100 Days Frontend Development Challenge**.

## 📌 Project Overview
A simple CSS keyframe animation project where a circular element moves horizontally across the screen while rotating continuously.

This project focuses on understanding CSS animations and keyframes.

## 🧩 Features
- Circular element with border styling
- Horizontal movement animation
- 360° rotation effect
- Infinite smooth animation
- Minimal dark theme design

## 🛠 Tech Stack
- HTML5
- CSS3
- CSS Keyframes Animation

## 🎯 What I Practiced
- @keyframes usage
- Position animation using left property
- Rotation using transform/rotate
- Infinite animation loop
- Linear timing function
