# Day 71 – JavaScript Toggle Button 🎮

Part of my **100 Days Frontend Development Challenge**.

## 📌 Project Overview

Built an interactive toggle button using JavaScript and CSS.

When the user clicks the button, JavaScript checks whether the button already contains the active CSS class. If not, it adds the class to change the button's appearance. Otherwise, it removes the class, creating a toggle effect.

This project demonstrates dynamic class manipulation using the DOM.

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)

## ✨ Features

- Interactive toggle button
- Dynamic class addition
- Dynamic class removal
- Active/Inactive button states
- Click event handling
- Modern button styling
- DOM manipulation
- Beginner-friendly project

## 📚 Concepts Practiced

- DOM Manipulation
- querySelector()
- classList
- classList.add()
- classList.remove()
- classList.contains()
- Conditional Statements
- Event Handling
- CSS Class Toggling

## 🚀 Learning Outcome

Learned how to dynamically change an element's appearance by adding and removing CSS classes using JavaScript, making web pages more interactive and responsive to user actions.
