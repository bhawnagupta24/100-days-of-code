## 📚 Concepts Practiced

### Math.random()

```javascript
const randomNumber = Math.random();
```

Returns a random number between:

```text
0 (inclusive)
and
1 (exclusive)
```

Example Output:

```javascript
0.2345
0.7821
0.9184
```
