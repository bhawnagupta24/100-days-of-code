# Day 33 – JavaScript Hover Product Interaction 📱✨

Part of my **100 Days Frontend Development Challenge**.

---

## 📌 Project Overview

An interactive product showcase page where hovering over different phone sections dynamically changes the background color and highlights the related heading.

The layout presents three smartphone brands:
- Apple iPhone
- Samsung Galaxy
- Nothing Phone

Each section responds to hover interactions using JavaScript event listeners.

## 🚀 How It Works

1. When the user hovers over a product box:
   - The background color of the main container changes.
   - The corresponding heading color updates.

2. When the mouse leaves the box:
   - The background resets to white.
   - The heading returns to its default color.

This interaction is handled using JavaScript event listeners.
