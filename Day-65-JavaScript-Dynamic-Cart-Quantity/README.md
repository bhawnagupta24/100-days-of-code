# Day 65 – JavaScript Dynamic Cart Quantity Display 🛒

Part of my **100 Days Frontend Development Challenge**.

## 📌 Project Overview

A dynamic Shopping Cart Quantity application built using HTML and JavaScript.

This project allows users to add or remove items from the cart while updating the quantity instantly on the webpage using JavaScript DOM manipulation. It also includes validation to prevent the cart quantity from exceeding the maximum limit or going below zero. :contentReference[oaicite:0]{index=0}

## 📸 Preview

This project demonstrates how JavaScript updates webpage content dynamically by displaying the current cart quantity whenever the user interacts with the cart buttons.

## 🚀 Features

- Show current cart quantity
- Add items to cart
- Increase quantity by +2, +3, +4, +5
- Remove items from cart
- Decrease quantity by -2 and -3
- Reset cart functionality
- Real-time quantity display
- Maximum cart limit validation
- Minimum cart limit validation
- DOM manipulation with JavaScript

## 🛠 Tech Stack

- HTML5
- JavaScript (ES6)

## 📚 Concepts Practiced

- DOM Manipulation
- querySelector()
- innerHTML
- Functions
- Function Parameters
- Conditional Statements
- Event Handling
- Dynamic Content Update
- Variables
- JavaScript Logic

## 🎯 Learning Outcome

Through this project, I learned how to dynamically update webpage content using JavaScript, manage application state with variables, create reusable functions, validate user actions, and improve user interaction through real-time DOM updates.
