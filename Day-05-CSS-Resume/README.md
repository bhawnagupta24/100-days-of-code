# Day 5 – Resume Design Using HTML & CSS

In this project, I created a personal resume using HTML and styled it using CSS.
The focus was on layout, typography, colors, and basic visual styling.

## What I Learned
- Structuring resume content using HTML
- Styling text using CSS (font-size, font-weight, letter-spacing)
- Using background colors and borders
- Applying border-radius for better UI
- Using IDs and selectors in CSS
- Linking external CSS to HTML

## Files Included
- resume.html – Resume structure using HTML
- resume.css – Styling for resume layout and design
