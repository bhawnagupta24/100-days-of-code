# Day 25 – Sliding Profile Card Animation 💙✨

Part of my **100 Days Frontend Development Challenge**.

## 📌 Project Overview
A dual-layer sliding card animation where the top icon card
moves upward and the bottom information card moves downward
on hover interaction.

## 🛠 Tech Stack
- HTML5
- CSS3
- Remix Icons

## ✨ Features
- Dual stacked card layout
- Smooth vertical slide animation
- Box shadow glow effect
- Hover-based interaction
- Icon centered layout
