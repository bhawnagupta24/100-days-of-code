# Day 59 – Heads or Tails Choice App 🎯

Part of my **100 Days Frontend Development Challenge**.

## 📌 Project Overview

A simple interactive Heads or Tails application built using HTML and JavaScript. Users can click either the **Heads** or **Tails** button, and the selected choice is instantly displayed on the screen using DOM manipulation.

## 🚀 Features

- Heads button
- Tails button
- Instant result display
- DOM manipulation
- Click event handling
- Beginner-friendly JavaScript logic
- Clean and minimal UI
- Lightweight project

## 🛠 Tech Stack

- HTML5
- JavaScript (ES6)

## 📚 Concepts Practiced

- DOM Selection
- onclick Events
- innerHTML
- Button Events
- User Interaction
- Dynamic Content Update

## 🎯 Learning Outcome

This project helped me understand how JavaScript can interact with HTML elements, respond to user actions, and dynamically update webpage content without refreshing the page.
