# Day 69 – CSS Button Styling Projects 🎨🖱️

Part of my **100 Days Frontend Development Challenge**.

## 📌 Project Overview

Built simple UI button components inspired by real-world applications like Amazon and Uber.

This project focuses on creating visually appealing buttons using CSS by applying colors, padding, border-radius, typography, spacing, and cursor effects to closely match modern website designs.

## 🛠 Tech Stack

- HTML5
- CSS3

## ✨ Features

### Amazon Button
- Amazon-style Add to Cart button
- Rounded pill-shaped design
- Brand-inspired yellow color
- Smooth and clean UI

### Uber Buttons
- Request Now button
- Schedule for Later button
- Modern minimal styling
- Proper spacing and typography

## 📚 Concepts Practiced

- CSS Colors
- Padding
- Border Radius
- Font Weight
- Cursor Property
- Margin & Spacing
- Button Styling
- UI Design Basics

## 🚀 Learning Outcome

Learned how to recreate real-world button components using only HTML and CSS while improving styling skills, spacing techniques, and modern UI design practices.
