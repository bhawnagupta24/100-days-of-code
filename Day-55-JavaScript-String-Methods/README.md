## 📚 Concepts Practiced

### toLowerCase()

Converts all characters of a string into lowercase.

```javascript
'Good Morning'.toLowerCase();
```

Output:

```text
good morning
```

---

### repeat()

Repeats a string a specified number of times.

```javascript
'hello'.repeat(2);
```

Output:

```text
hellohello
```

Example:

```javascript
'test'.repeat(3);
```

Output:

```text
testtesttest
```

## 💻 Program Logic

### Convert Text to Lowercase
