## 📚 Concepts Practiced

### Universal Selector

```css
* {
  box-sizing: border-box;
}
```

Provides consistent sizing for all elements.

---

### Full Page Layout

```css
html,
body {
  height: 100%;
  width: 100%;
}
```

Makes the layout occupy the full browser window.

---

### Background Styling

```css
background-color: beige;
```

Applies a clean and soft background color.

---

### Typography

```css
font-family:
'Franklin Gothic Medium',
'Arial Narrow',
Arial,
sans-serif;
```

Improves text readability.

---

### Button Styling

```css
button {
  padding: 10px 10px;
  border-radius: 20px;
  border: 1px solid black;
}
```

Creates rounded and visually appealing buttons.
