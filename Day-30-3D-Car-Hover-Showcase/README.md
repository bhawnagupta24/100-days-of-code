# Day 30 – 3D Car Hover Showcase 🚗✨

Part of my **100 Days Frontend Development Challenge**.

## 📌 Project Overview
A 3D interactive car showcase built using CSS perspective and rotateY effects.  
Each car card flips slightly in 3D space and reveals a secondary image with a smooth zoom transition on hover.

This project focuses on advanced hover interactions and 3D transformations.

---

## 🎯 Features
- 3 Car interactive cards
- CSS perspective effect
- rotateY 3D transformation
- Image swap on hover
- Smooth opacity transition
- Zoom-in hover animation
- Clean layout with flexbox
- Fully responsive centered design

---

## 🧠 What I Practiced
- Using `perspective` for 3D depth
- Applying `transform: rotateY()`
- Image overlay positioning with absolute
- Opacity transitions
- Hover-based animations
- Layer control using positioning
- Clean card structure design

---

## 🛠 Tech Stack
- HTML5
- CSS3
- Flexbox
- CSS Transforms
- CSS Transitions
