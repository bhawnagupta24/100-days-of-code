## 📚 Concepts Practiced

### Creating an Object

```javascript
const product = {
  name: 'basketball',
  price: 2095
};
```

---

### Accessing Properties

```javascript
product.name
product.price
```

---

### Updating Properties

```javascript
product.price += 500;
```

---

### Adding New Properties

```javascript
product['delivery-time'] = '3 days';
```

Bracket notation is useful when property names contain special characters such as hyphens.
