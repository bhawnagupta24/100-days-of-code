# Day 40 – JavaScript Greeting System ☀️🌙

Part of my **100 Days Frontend Development Challenge**.

---

## 📌 Project Overview

A simple JavaScript project that displays different greeting messages based on the current time of the day.

The program uses conditional statements to determine whether it is morning, afternoon, or night and then displays a personalized greeting message.

---

## 🎯 Features

- Time-based greetings
- Personalized user name display
- Conditional logic using if-else statements
- Template literals
- Beginner-friendly JavaScript concepts

---

## 🛠 Tech Stack

- HTML5
- JavaScript (ES6)

## 💻 Program Logic

Morning:
- 6 AM to 12 PM

Afternoon:
- 1 PM to 5 PM

Night:
- Remaining hours

The program checks the current hour and prints the appropriate greeting message.
