# Day 4 – HTML Forms Practice

In this project, I practiced creating HTML forms and explored different input types along with basic CSS styling.

## What I Learned
- How to create forms using the `<form>` tag
- Using different input types:
  - text, email, number, password
  - radio buttons, checkboxes
  - date, color, range, file, search
- Using `<label>`, `<select>`, `<option>`, and `<textarea>`
- Difference between basic form and styled form
- Applying basic CSS for layout and positioning

## Files Included
- `form.html` – Basic HTML registration form
- `form2.html` – HTML form with CSS styling
- `form2.css` – Basic CSS for form layout and design
