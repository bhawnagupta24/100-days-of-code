# Day 24 – 2026 Interactive Grid Animation 🎉✨

Part of my **100 Days Frontend Development Challenge**.

## 📌 Project Overview
A creative 4-grid image layout with hover-based directional animation.
When hovering over the container, images shift diagonally creating
a dynamic split-grid effect.

## 🛠 Tech Stack
- HTML5
- CSS3

## ✨ Features
- 4 quadrant image layout
- Diagonal hover translation animation
- Smooth transition effects
- Button hover styling
- Centered flexbox layout
