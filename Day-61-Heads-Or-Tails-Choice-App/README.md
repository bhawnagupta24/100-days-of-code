# Day 61 – Heads or Tails Choice App 🎯

Part of my **100 Days Frontend Development Challenge**.

## 📌 Project Overview

A simple interactive Heads or Tails application built using **HTML** and **JavaScript**. Users can click either the **Heads** or **Tails** button, and the selected result is instantly displayed on the screen using JavaScript DOM manipulation.

## 📸 Preview

This mini project demonstrates how JavaScript updates webpage content dynamically based on user interaction without reloading the page.

## 🚀 Features

- Interactive Heads & Tails buttons
- Instant result display
- DOM manipulation using JavaScript
- Click event handling
- Clean and beginner-friendly UI
- Lightweight and responsive layout
- Real-time text update
- Easy to understand project structure

## 🛠 Tech Stack

- HTML5
- JavaScript (ES6)

## 📚 Concepts Practiced

- DOM Manipulation
- onclick Events
- querySelector()
- innerHTML
- Event Handling
- Dynamic Content Update
- User Interaction
- JavaScript Basics

## 🎯 Learning Outcome

Through this project, I learned how JavaScript interacts with HTML elements, handles user events, and dynamically updates webpage content using DOM manipulation.
