# Day 75 – JavaScript Shipping Calculator with Validation 🛒✅

Part of my **100 Days Frontend Development Challenge**.

## 📌 Project Overview

Built an enhanced Amazon Shipping Calculator that validates user input before calculating the final order cost.

The application checks whether the entered cost is valid. If a negative value is entered, it displays an error message instead of calculating the total. For valid orders below the free shipping threshold, shipping charges are added automatically, while higher-value orders receive free shipping.

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)

## ✨ Features

- Amazon-style shipping calculator
- User input validation
- Error message for negative values
- Automatic shipping charge calculation
- Free shipping for eligible orders
- Enter key support
- Dynamic DOM updates
- Responsive and clean UI

## 📚 Concepts Practiced

- DOM Manipulation
- querySelector()
- Event Handling
- Keyboard Events
- Conditional Statements
- Input Validation
- Number Conversion
- Dynamic Error Handling
- innerHTML Updates

## 🚀 Learning Outcome

Learned how to validate user input, handle invalid data gracefully, and build a more reliable JavaScript application by combining conditional logic, DOM manipulation, and dynamic error messaging.
