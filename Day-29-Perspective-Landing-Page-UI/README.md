# Day 29 – Perspective Landing Page UI 🚀✨

Part of my **100 Days Frontend Development Challenge**.

## 📌 Project Overview
A modern SaaS-style landing page UI inspired by dashboard platforms.  
This project focuses on clean layout structuring, navigation design, CTA buttons, and perspective-based image styling.

## 🧩 Features
- Clean top navigation bar
- Logo branding section
- Centered hero content
- Call-to-action buttons
- Watch demo button with Remix Icon
- Perspective styled hero image
- Notification badge design
- Minimal modern layout

## 🛠 Tech Stack
- HTML5
- CSS3
- Flexbox
- Remix Icons

## 🎯 What I Practiced
- Navbar layout alignment
- Flexbox structuring
- CTA button styling
- Perspective & rotateX effect
- Positioning with absolute elements
- Clean SaaS landing page layout design
