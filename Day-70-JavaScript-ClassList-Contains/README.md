# Day 70 – JavaScript classList.contains() 🔍

Part of my **100 Days Frontend Development Challenge**.

## 📌 Project Overview

A beginner-friendly JavaScript project focused on checking whether an HTML element contains a specific CSS class using the `classList.contains()` method.

This project demonstrates how JavaScript can inspect DOM elements and verify the existence of CSS classes, which is useful for creating interactive web applications and implementing conditional UI behavior.

## 🛠 Tech Stack

- HTML5
- JavaScript (ES6)

## ✨ Features

- Select HTML elements using `querySelector()`
- Check for a specific CSS class
- Display Boolean (`true` / `false`) result in the console
- Simple DOM interaction
- Beginner-friendly JavaScript example

## 📚 Concepts Practiced

- DOM Manipulation
- querySelector()
- classList
- classList.contains()
- Boolean Values
- Console Output

## 🚀 Learning Outcome

Learned how to inspect HTML elements, verify CSS classes using JavaScript, and understand how `classList.contains()` helps in building interactive UI components and conditional styling.
