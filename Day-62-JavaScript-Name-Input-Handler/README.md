# Day 62 – JavaScript Name Input & Enter Key Handler ⌨️

Part of my **100 Days Frontend Development Challenge**.

## 📌 Project Overview

A simple interactive JavaScript project that captures the user's name from an input field and displays it dynamically on the webpage.

The project also improves the user experience by allowing the **Enter** key to submit the input without clicking the button. Finally, the repeated logic is refactored into a reusable function.

This project focuses on DOM manipulation, keyboard events, functions, and user interaction. :contentReference[oaicite:0]{index=0}

## 📸 Preview

This mini project demonstrates how JavaScript reads user input, updates webpage content dynamically, and handles keyboard events for smoother interaction.
