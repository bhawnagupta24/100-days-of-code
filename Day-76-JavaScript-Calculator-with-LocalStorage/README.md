# Day 76 – JavaScript Calculator with Local Storage 🧮

Part of my **100 Days Frontend Development Challenge**.

## 📌 Project Overview

Built a fully functional calculator using HTML, CSS, and JavaScript with a clean, modern interface.

The calculator supports basic arithmetic operations, dynamically updates the calculation display, evaluates expressions, and stores the current calculation in Local Storage. This allows users to continue their calculations even after refreshing the browser.

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage

## ✨ Features

- Modern calculator UI
- Addition, Subtraction, Multiplication, and Division
- Live calculation display
- Decimal number support
- Evaluate expressions
- Clear calculator functionality
- Local Storage persistence
- Automatically restores previous calculation after page refresh
- Responsive button layout

## 📚 Concepts Practiced

- DOM Manipulation
- querySelector()
- Functions
- Event Handling
- Local Storage
- eval()
- String Manipulation
- Dynamic UI Updates
- Arithmetic Operations

## 🚀 Learning Outcome

Learned how to build a complete calculator application by combining DOM manipulation, reusable functions, Local Storage, dynamic rendering, and arithmetic logic to create a real-world interactive JavaScript project.
