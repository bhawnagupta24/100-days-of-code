## 📚 Concepts Practiced

### Function Declaration

```javascript
function greet() {
  console.log('Hello!');
}
```

### Function Call

```javascript
greet();
```

### Parameters

```javascript
function greet(name) {
  console.log(`Hello ${name}`);
}
```

### Arguments

```javascript
greet('Bhawna');
```

### Conditional Statements

```javascript
if (!name) {
  console.log('Hi there!');
}
```

---

## 💻 Program Logic

### Basic Greeting

```javascript
function greet() {
  console.log('Hello!');
}
```

### Personalized Greeting

```javascript
function greet(name) {
  console.log(`Hello ${name}`);
}
```

### Handle Missing Name

```javascript
function greet(name) {
  if (!name) {
    console.log('Hi there!');
  } else {
    console.log(`Hello ${name}!`);
  }
}
```
