# Day 39 – GSAP Personal Poetry Showcase ✨📖

Part of my **100 Days Frontend Development Challenge**.

---

## 📌 Project Overview

A creative personal portfolio page built using **HTML, CSS, JavaScript, and GSAP** animations.

This project showcases my poems, writings, books that inspire me, and my love for storytelling through an elegant animated interface.

The page introduces visitors with smooth entrance animations, split-text effects, image transitions, and scroll-based interactions.
