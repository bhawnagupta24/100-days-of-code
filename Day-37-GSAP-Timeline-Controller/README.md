# Day 37 – GSAP Timeline Controller 🎬⚡

Part of my **100 Days Frontend Development Challenge**.

---

## 📌 Project Overview

This project demonstrates advanced **GSAP Timeline Controls** using JavaScript.

Three animated squares perform smooth rotation and transformation effects while different control buttons allow users to:

- Play animations
- Pause animations
- Restart timeline
- Reverse animation
- Resume animation
- Control animation speed

This project helped me understand how animation timelines work in GSAP.
