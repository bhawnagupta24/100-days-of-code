# Day 74 – JavaScript Single Active Toggle Buttons 🎮🎵💻

Part of my **100 Days Frontend Development Challenge**.

## 📌 Project Overview

Built an improved category toggle button interface where only one button can remain active at a time.

Unlike the previous project, selecting a new category automatically removes the active state from the previously selected button. This behavior is achieved using reusable JavaScript functions and dynamic DOM manipulation, similar to category selectors used in modern web applications.

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)

## ✨ Features

- Gaming, Music, and Tech category buttons
- Only one active button at a time
- Automatically deselects the previous button
- Reusable toggle function
- Dynamic CSS class management
- External CSS and JavaScript files
- Clean and maintainable code structure
- Interactive user interface

## 📚 Concepts Practiced

- DOM Manipulation
- querySelector()
- classList.add()
- classList.remove()
- classList.contains()
- Reusable Functions
- Function Parameters
- Conditional Statements
- State Management
- External CSS & JavaScript Files

## 🚀 Learning Outcome

Learned how to manage the active state of multiple UI elements by ensuring only one button remains selected at a time. This project strengthened my understanding of reusable functions, DOM manipulation, and state management techniques commonly used in modern frontend development.
