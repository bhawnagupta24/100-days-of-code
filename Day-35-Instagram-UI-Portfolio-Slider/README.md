# Day 35 – Instagram UI Portfolio Slider 📱✨

Part of my **100 Days Frontend Development Challenge**.

---

## 📌 Project Overview

This project is an **Instagram-inspired portfolio interface** built using HTML, CSS, and JavaScript with the Swiper.js library.

It displays different sections of an Instagram-like interface such as profile, posts, reels, feed, notifications and messages using a **card slider animation**.

Users can navigate between the cards using arrow buttons.

---

## 🎯 Features

- Instagram-style navigation bar
- Profile introduction section
- Card slider interface
- Swiper.js card animation effect
- Custom left and right navigation arrows
- Portfolio showcase layout
- Clean UI design

---

## 📱 Sections Included

The slider shows different UI screens:

- Profile
- Instagram Feed
- Reels
- Notifications
- Messages
- Posts

Each card represents a different section of the Instagram interface.

## 🚀 How It Works

1. Swiper.js creates a **card slider interface**.
2. Each slide contains an Instagram UI screenshot.
3. Left and right arrow buttons allow navigation between slides.
4. The **cards effect** gives a stacked UI animation.
