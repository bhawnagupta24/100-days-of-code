# Day 34 – Mouse Follow Achievements Showcase 🏆✨

Part of my **100 Days Frontend Development Challenge**.

---

## 📌 Project Overview

An interactive achievements showcase where images follow the mouse cursor and change dynamically when hovering over different headings.

Each heading represents one of my achievements, and when the user moves the cursor over a heading, the corresponding certificate image appears inside a floating container.

This creates a modern portfolio-style interaction.

## 🚀 How It Works

1. When the user moves the mouse inside the page,
   JavaScript tracks the cursor position.

2. The image container moves with the cursor.

3. When hovering over different headings,
   the image wrapper translates vertically to reveal the correct image.
