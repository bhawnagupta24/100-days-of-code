# Day 38 – JavaScript Modules & Rest Parameters 📦⚡

Part of my **100 Days Frontend Development Challenge**.

---

## 📌 Project Overview

This project demonstrates the use of:

- JavaScript ES6 Modules
- Import & Export functionality
- Rest Parameters
- Array iteration using `forEach()`

A reusable `add()` function is exported from one JavaScript file and imported into another file to calculate the total sum of multiple numbers dynamically.

---

## 🎯 Features

- ES6 Module system
- Export default function
- Importing modules

- Dynamic addition of values
- `forEach()` loop implementation
- Console-based output
