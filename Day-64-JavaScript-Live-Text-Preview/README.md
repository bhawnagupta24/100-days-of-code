# Day 64 – JavaScript Live Text Preview ⌨️

Part of my **100 Days Frontend Development Challenge**.

## 📌 Project Overview

A simple JavaScript project that displays the user's input in real time as they type.

The application listens for keyboard input using the **onkeyup** event and immediately updates the webpage using JavaScript DOM manipulation, creating a live text preview experience. :contentReference[oaicite:0]{index=0}

## 📸 Preview

This project demonstrates how JavaScript captures every keystroke from an input field and dynamically updates the displayed text without refreshing the page.
