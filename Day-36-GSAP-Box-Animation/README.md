# Day 36 – GSAP Box Animation 🎞️✨

Part of my **100 Days Frontend Development Challenge**.

---

## 📌 Project Overview

This project demonstrates a simple animation using the **GSAP (GreenSock Animation Platform)** library.

A box element moves horizontally across the screen while rotating 360 degrees.  
The animation repeats multiple times creating a smooth motion effect.

This project introduces the basics of **GSAP animations and timeline control**.

## 🎯 Features

- Smooth animation using GSAP
- Horizontal movement animation
- 360° rotation effect
- Repeating animation loop
- Clean minimal UI
- External animation library integration

## 🚀 How It Works

1. GSAP library is imported using CDN.
2. JavaScript targets the `.box` element.
3. GSAP animates the box:

- Moves it horizontally
- Rotates it 360 degrees
- Runs for 3 seconds
- Repeats 5 times
