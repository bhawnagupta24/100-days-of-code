# Day 73 – JavaScript Reusable Toggle Buttons 🔘

Part of my **100 Days Frontend Development Challenge**.

## 📌 Project Overview

Built a reusable toggle button system using HTML, CSS, and JavaScript.

Instead of writing separate logic for each button, this project uses a single reusable `toggleButton()` function that accepts a CSS selector and toggles the active state of any button. The project is also organized into separate HTML, CSS, and JavaScript files, making the code cleaner and easier to maintain.

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)

## ✨ Features

- Reusable toggle function
- Gaming, Music, and Tech buttons
- Dynamic active/inactive states
- Independent button toggling
- External CSS and JavaScript files
- Cleaner and modular code structure
- Interactive UI

## 📚 Concepts Practiced

- Functions with Parameters
- DOM Manipulation

- classList.contains()
- Reusable Functions
- Code Reusability

## 🚀 Learning Outcome

Learned how to write reusable JavaScript functions that work for multiple elements, reducing code duplication while improving project organization, readability, and maintainability.
