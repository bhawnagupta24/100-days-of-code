# Day 77 – JavaScript Modular Calculator 🧮

Part of my **100 Days Frontend Development Challenge**.

## 📌 Project Overview

Built a modular calculator by separating the project into dedicated HTML, CSS, and JavaScript files.

The calculator performs basic arithmetic operations, updates the calculation dynamically, and stores the latest calculation using Local Storage. By moving the styling and logic into external files, the project becomes cleaner, more organized, and easier to maintain.

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage

## ✨ Features

- Modular project structure
- External CSS and JavaScript files
- Basic arithmetic operations
- Live calculation display
- Decimal number support
- Expression evaluation
- Clear button functionality
- Local Storage persistence
- Automatically restores previous calculation after page refresh
- Clean and responsive calculator UI

## 📚 Concepts Practiced

- DOM Manipulation
- querySelector()
- Functions
- External JavaScript Files
- External CSS Files
- Local Storage
- eval()
- Dynamic UI Updates
- Code Organization
- Separation of Concerns

## 🚀 Learning Outcome

Learned how to organize a JavaScript application into separate HTML, CSS, and JavaScript files while maintaining functionality. This project improved code readability, maintainability, and reinforced best practices for frontend development.
