# Day 66 – JavaScript Calculator with Live Display 🧮

Part of my **100 Days Frontend Development Challenge**.

## 📌 Project Overview

A fully interactive calculator built using HTML and JavaScript.

This version improves the previous calculator by displaying the current mathematical expression on the webpage in real time. The application also stores the calculation using **Local Storage**, allowing users to continue where they left off even after refreshing the browser.

This project focuses on DOM manipulation, reusable functions, browser storage, event handling, and dynamic UI updates. :contentReference[oaicite:0]{index=0}

## 📸 Preview

This project demonstrates how JavaScript builds mathematical expressions dynamically, displays them instantly on the webpage, evaluates calculations, and saves the current state using Local Storage.

## 🚀 Features

- Interactive calculator interface
- Live calculation display
- Number buttons (0–9)
- Arithmetic operators (+, -, ×, ÷)
- Decimal support
- Evaluate expressions
- Clear calculator
- Local Storage integration
- Automatic calculation recovery
- Real-time DOM updates

## 🛠 Tech Stack

- HTML5
- JavaScript (ES6)

## 🎯 Learning Outcome

Through this project, I learned how to build a dynamic calculator interface, update webpage content in real time, store application data using Local Storage, and create reusable JavaScript functions for better code organization.
